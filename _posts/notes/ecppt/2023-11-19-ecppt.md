- Tips

Not required to pass the exam:
```
- System Security
- Powershell
- Wi-fi
- Ruby
```

However, Be prepared to: 
```
- Buffer Overflow
- Pivoting (learn Chisel, its gonna be useful in the long run)
- Web app
- Understand the Network (drawing network maps helps)
```
